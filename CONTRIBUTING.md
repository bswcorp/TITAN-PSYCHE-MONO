# Contributing to TITAN-PSYCHE-MONO

Thank you for contributing. To protect project IP and ensure smooth collaboration, please follow these steps:

1. Sign the Contributor License Agreement (CLA)
   - Add a signed statement in your PR description:
     "I, [Full Name], confirm that I am the author of the attached Contribution and accept the terms of the CLA."
   - Attach signed CLA to the PR if requested.

2. Code organization
   - Public modules: place in /core or /modules
   - Proprietary modules: place in /enterprise or /proprietary and include license header referencing LICENSE_COMMERCIAL.md

3. PR checklist
   - Include tests and documentation
   - Confirm CLA signature statement present
   - Label PR with `enterprise` if it touches proprietary code

4. Security
   - Report vulnerabilities to security@stg-sovereign.com
